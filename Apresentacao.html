✈️ Férias Aurora, Ana e Allan - Planejador Mágico de Viagens

Bem-vindo(a) à apresentação oficial do nosso Planejador de Viagens!

Este aplicativo foi idealizado e construído sob medida para garantir que o planejamento das férias em família seja não apenas organizado, mas incrivelmente divertido e inteligente. Ele é uma aplicação web interativa que funciona como um "Assistente de Viagens Pessoal", cruzando dados em tempo real para montar o cenário perfeito para qualquer destino do mundo.

🌟 O que exatamente este aplicativo faz?

Imagine não precisar abrir 10 abas diferentes para calcular câmbio, ver hotéis, montar roteiros, fazer checklists de malas e anotar custos. Este aplicativo centraliza exatamente tudo:

  • Roteiros Inteligentes (Itinerário Dinâmico): Você diz para onde vai, quantos dias e quem vai com você. O aplicativo gera uma linha do tempo dia após dia. Vai levar bebês ou crianças (Aurora)? Ele automaticamente substitui programas cansativos por "Parques com Área Kids" e cria roteiros mais espaçados.
  • Câmbio e Orçamento em Tempo Real: Digitou que o destino é "Paris"? O aplicativo percebe, muda a moeda oficial para Euro (EUR) e busca a cotação exata ao vivo. Tudo o que você registrar de gastos na viagem é convertido magicamente para Reais no painel financeiro.
  • Sugestões de Restaurantes e Hotéis Contextuais: O app gera opções incríveis de locais que combinam com as preferências de dieta do grupo (ex: Vegetariana, Alta Gastronomia). Gostou de um? Clique em "Ver no Mapa" ou "Reservar" e ele abre o Google Maps direto no endereço.
  • Malas Organizadas (Checklist Duplo): Um checklist interativo em duas colunas: uma aba com itens de sobrevivência para o bebê (fórmula, fraldas, babá eletrônica) e outra aba com itens dos adultos e lembretes de casa (Passaportes, Trancar janelas, Ligar alarme).
  • Simulador de Transportes: Integração nativa para busca instantânea no Google Flights e calculadoras de gastos previstos de Uber/Táxi ou Aluguel de Carro no destino escolhido.

🛠️ Tecnologias Utilizadas

Para quem tem curiosidade sobre como isso funciona nos bastidores, o software conta com tecnologias avançadas do mercado:

  • Next.js 15+ & React: O motor principal da aplicação, garantindo telas instantâneas sem travamentos.
  • Tailwind CSS: Para criar uma interface limpa, cantos arredondados, efeitos "glassmorphism" (vidro) e ser 100% responsivo.
  • Inteligência e Dados Rest: Uso de Context API para lembrar o status da última busca conectada a uma API financeira dinâmica de moedas.
